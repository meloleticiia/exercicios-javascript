# exercicios-javascript

Coleção de exercícios práticos para desenvolvimento com JavaScript.

✅ Pré-requisitos

Antes de começar, verifique se você tem instalado:
- Node.js (versão 18 ou superior)
- npm (vem com o Node.js)

🔍 **Como verificar as instalações**:
```bash
node -v
npm -v

📁 Como baixar o projeto:

git clone https://github.com/meloleticia/exercícios.javascript.git
cd exercícios.javascript

▶️ Como executar os exercícios:

Abra o terminal (ou o terminal integrado do VS Code).

Navegue até a pasta onde está o exercício. Exemplo:

cd exercicios-fullstack

Execute o arquivo desejado com o Node.js. Por exemplo:

node atividade1.js
